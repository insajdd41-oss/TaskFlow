# Планировщик задач

## Описание
Веб-приложение для управления задачами. Можно добавлять, выполнять, удалять и редактировать задачи. Всё сохраняется в браузере.

## Что умеет
- Добавлять и удалять задачи
- Отмечать выполненные
- Фильтровать (все/активные/выполненные)
- Считать статистику (продуктивность, время)
- Рисовать диаграмму за неделю
- Давать достижения за выполнение
- Тёмная тема
- 3D-кубик

## Технологии
- HTML
- CSS
- JavaScript
- localStorage
