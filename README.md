# Планировщик задач

## Описание проекта
Веб-приложение для управления задачами с возможностью добавления, выполнения и удаления задач.

## Функционал
- ✅ Добавление/удаление задач
- ✅ Отметка о выполнении
- ✅ Фильтрация (все/активные/выполненные)
- ✅ Статистика продуктивности
- ✅ Сохранение в localStorage
- ✅ Адаптивный дизайн
- ✅ 3D-анимация

## Технологии
- HTML5
- CSS3 (Flexbox, Grid, анимации, 3D)
- JavaScript (ES6)
- LocalStorage
